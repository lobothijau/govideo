Uses

This project was meant to be simple and easy to use. I build it for my own learning to help me implement a web app in Go programming language. I hope one day it can help me land a job as a Go developer while being useful for anyone who might need them.

I use the following stack:

  • Go without any framework
  • MySQL for the database
  • TailwindCSS for styling the UI
  • jQuery & simple JavaScript for the frontend (sorry)
  • DigitalOcean for hosting
